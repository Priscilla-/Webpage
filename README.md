# GitHub Pages-ready academic website

This is a simple static website that can be uploaded directly to a GitHub repository and published with GitHub Pages.

## Files

- `index.html`
- `about/index.html`
- `research/index.html`
- `publications/index.html`
- `teaching/index.html`
- `contact/index.html`
- `assets/css/style.css`

## How to publish on GitHub Pages

1. Create a new repository on GitHub.
2. Upload all files in this folder.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and the `/ (root)` folder.
6. Save.
7. GitHub will publish the site at your GitHub Pages URL.

## Notes

- The site uses only relative links, so it works both for a user site (`yourusername.github.io`) and for a project site (`yourusername.github.io/repository-name/`).
- Replace placeholder text, profile links, and contact details.
- Add a profile photo at `assets/images/profile.jpg` if desired.
